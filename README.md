# 🧠 Evaluación de Autocompasión

Una aplicación interactiva basada en la prueba de Autocompasión que mide cómo te tratas a ti mismo en momentos difíciles. Ofrece resultados personalizados en tres niveles: Alta, Moderada o Baja autocompasión y te ayuda a reflexionar sobre cómo mejorar.

🔗 **[Ver Demo en Vivo](https://sumamara.github.io/Encuesta_autocompasion/)**

## 🎯 Qué evaluamos

El test incluye 12 afirmaciones puntuadas del 1 (Casi nunca) al 6 (Siempre), de las cuales algunas miden aspectos positivos directos (mindfulness, bondad humana compartida) y otras tienen puntuación inversa (aislamiento, autocrítica severa) para equilibrar el test.

### Niveles Resultantes
| Puntuación (12-72) | Nivel de Autocompasión |
|---|---|
| **43 - 72** | 🟢 Alta |
| **31 - 42** | 🟡 Moderada |
| **12 - 30** | 🔴 Baja |

## 🌟 Características

- **12 preguntas focalizadas** con escala de respuestas de 6 puntos.
- **Lógica experta de puntuación** — Ajuste automático (inverso) de puntos en ciertas preguntas negativas para garantizar precisión en la escala final.
- **Gráfico de Resultados Dinámico** — Barra de progreso vertical e indicadores visuales según las zonas de puntuación.
- **Resultados accionables** — Desglose de información con consejos prácticos (ej."Trátate como a un amigo").
- **Persistencia de datos** integrada de manera limpia con Supabase.
- **Diseño responsivo** — Experiencia impecable en celular, tablet y escritorio con base a Shadcn UI y Tailwind.

## 🛠️ Tecnologías

- **Frontend**: Next.js (React), TypeScript
- **Estilos**: Tailwind CSS, Shadcn UI, Lucide React
- **Backend**: Supabase
- **Deploy**: GitHub / Vercel

## 🚀 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Sumamara/Encuesta_autocompasion_nextjs.git
cd Encuesta_autocompasion_nextjs

# Instalar dependencias
npm install

# Configurar variables de entorno
# Crear archivo .env o .env.local con:
# NEXT_PUBLIC_SUPABASE_URL=tu_url
# NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_key

# Ejecutar el modo desarrollo
npm run dev
```

## 📊 Base de Datos (Supabase)

Estructura de la tabla preparada en SQL:

```sql
CREATE TABLE QuizAutocompasion (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  user_name text,
  total_score integer not null,
  time_taken_seconds integer,
  q1 integer, q2 integer, q3 integer, q4 integer,
  q5 integer, q6 integer, q7 integer, q8 integer,
  q9 integer, q10 integer, q11 integer, q12 integer
);

-- Políticas RLS
ALTER TABLE QuizAutocompasion ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Permitir inserción pública" ON QuizAutocompasion FOR INSERT TO public WITH CHECK (true);
```
